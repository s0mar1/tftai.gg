## 💡 사용 가이드라인

### 색상 사용 원칙
1. **의미론적 사용**: 색상은 기능과 의미에 맞게 사용합니다.
2. **일관성 유지**: 같은 기능은 항상 같은 색상을 사용합니다.
3. **접근성 고려**: WCAG 2.1 AA 기준의 색상 대비를 유지합니다.

### 타이포그래피 원칙
1. **계층 구조**: 명확한 시각적 계층을 만듭니다.
2. **가독성 우선**: 한글 가독성을 최우선으로 고려합니다.
3. **일관된 크기**: 정의된 크기만 사용합니다.

### 간격 사용 원칙
1. **8px 그리드**: 8의 배수를 기본으로 사용합니다.
2. **일관성**: 같은 맥락에서는 같은 간격을 사용합니다.
3. **반응형 고려**: 화면 크기에 따라 적절히 조정합니다.
